---

## 📌 О проекте

**HackatonUI** — это информационная система поддержки принятия решений, предназначенная для:

- контроля объектов (зданий)
- управления задачами и поручениями
- фиксации решений
- мониторинга сроков исполнения
- централизованного хранения данных

Проект разработан в рамках учебной/практической работы.

## ⚙️ Основные возможности

### 🏢 Управление объектами
- создание и редактирование зданий  
- удаление объектов  
- гибкая настройка атрибутов  
- поиск по полям

### 📋 Задачи и решения
- создание задач для объектов

- роли: **Admin / User**

## 🏗️ Архитектура

Проект реализован по паттерну **MVVM**:

- **View** — XAML интерфейс  
- **ViewModel** — логика и биндинги  
- **Model** — данные  

### Структура проекта

```
HackatonUI/
├── Models/
├── ViewModels/
├── Views/
├── Repositories/
├── Services/
├── Converters/
├── Data/
└── App.axaml
```
